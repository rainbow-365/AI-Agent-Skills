# AI Agent Skills

This repository contains a collection of skills for AI agents. These skills extend the capabilities of agents, allowing them to perform specialized tasks such as cybersecurity reviews, code analysis, and more.

## Available Skills

### [Cyber-Review](skills/cyber_review/SKILL.md)
An autonomous cybersecurity reviewer that scans codebases for vulnerabilities, secrets, and privacy risks.

**Capabilities:**
*   **Project Reconnaissance**: Identify language, framework, and dependencies.
*   **Secret Scanning**: Detect hardcoded keys, passwords, and tokens.
*   **Code Analysis**: Find dangerous functions and insecure patterns.
*   **Reporting**: Generate professional security audit reports.
